# Bug Report – Pagamento não concluído

Título:
Pagamento não é concluído após clicar em "Pagar"

Ambiente:
Chrome | Windows 10

Descrição:
Ao tentar realizar o pagamento com dados corretos, o pagamento não é concluído.

Passos para reproduzir:
1. Acessar página de pagamento  
2. Inserir dados corretos  
3. Clicar em "Pagar"  

Resultado atual:
Pagamento não é concluído e nenhuma confirmação é exibida.

Resultado esperado:
Pagamento deveria ser concluído com sucesso.
