# QA Portfolio – Richard Moraes

## Projeto: Sistema de E-commerce (Simulado)

Objetivo:
Simular testes de qualidade em um sistema fictício de e-commerce.

Escopo:
- Login
- Cadastro
- Pagamento
- Recuperação de senha

Atividades realizadas:
- Criação de casos de teste
- Execução de testes funcionais e exploratórios
- Elaboração de bug reports

Ferramentas:
- Testes manuais
- Documentação em GitHub
- Navegador: Chrome | Windows 10
